Lesson 2.1: Introduction to Serious Programming

Computer

Unlike a toaster that can basically do one thing toast bread pastries etc... Computers can do anything with programming. Without any programs computers are useless.

Program

Programs are what make a computer do stuff.

Think like a programmer: think procedurally, be an abstract thinker, be systems thinker, have technological empathy, and be a debugger.

Programming Languages

There are many different types of programming languages and the one we will be using is python. The reason why we do not use a language such as english or spanish is because of ambiguity and verbosity. Computers need precise step by step instruction to do what you want them to do.

Interpreter

Python uses an interpreter to do what we want the program to do.

Python

Python is a high level language meaning they are closer to the human languages than a machine language.

Grammer

Python uses Backus Naur form. Where a non-terminal is like a noun, verb, object, subject etc... A terminal would be python, eat, etc...

Python Expresion

Python expresion can be in the form of 1 * 1, 1 - 4, 1+3 etc... Using the Backus Naur form a non-terminal would be expression, operator and the terminal would be 1, -, + () etc...

Lesson 2.2: Variables and Strings

How do you assign a value to a variable in Python?

Name = Expression

What is the difference in the = in programming then in regular math?

In programming the = is usually used to assign a value.

How can variables make you life easier as a programmer?

There are many reasons to use variables here are a few: easier to identify what numbers mean in expression, making for easier to read code. Makes it so you can use that data in other places in your code.

How is a string identified in Python?

A string in Python is identified with single quotes 'string' or double quotes "string".

What are some string functions/methods/procedures in Python?

Find is used to find the position of a character and returns the position number.
<string>.find(<string>)
ex1: 'string'.find('s') # this returns 0
ex2: 'string'.find('g') # this returns 5
ex3: 'string'.find('a') # returns -1 if it cannot find
<string>.find(<string>,<number>)
ex4: 'strings'.find('s',1) # starts the search at position 1 this returns 6

Lesson 2.3: Input->Function->Output

What are functions?

Functions are blocks of code that allow the programmer to do particular job that can be done over and over...

How do you make and use a function?

In Python you use the following format to make a function:
def name(par1, par2...):
    par1 = par1 + par1
    return par1
To call a function you use the following:
name(par1,par2)

How do functions help programmers avoid repetition?

Functions are great because they only have to write them once and can reuse them over and over again throughout the program.

What is a return statement in a function?

The return statement in a function allows you to send something to the program for future use.

Lesson 2.4: Control Flow & Loops: if and while

What is an if Statements and how do if statements work?

An if statement is a control that is either True or False. An if statemene helps you make decisions on what should or shouldn't be done next. An if statement works by passing it in a value which can be a bolean value or and equation the evaluates to a bolean value.

What is a while statement and how does the while statements work?

A while statement allows to do something over and over until a certain condition is met. A while statement consist of a condition or expression and a block of code to be executed until the condition is met.

Lesson 3: Program with Objects

OOP - Object Oriented Programming

OOP is based on object that can be used over and over. The ability to be able to write a blueprint(class) and then use that blueprint to duplicate objects. OOP makes it easier to reuse code and share with others.

Important Terms

There are many important terms in programming here are a few:

abstraction
In programming abstraction is the ability to use a function or method that already exist and create something from it. For example adding a movie to a website you can use a method from a lets say a movie class to add that movie to the website. We do not need the details of function/method to use it.
class
A class is like a blueprint and allows you to create multiple objects.
instance
Creating an object from a class is an instance of that class.
constructor
In python a constructor (__init__) is used to construct/initialize a class.
self
Is used when constructing a class instance.
instance variables
Are used to assign properties/fields to a class.
import
Is used to bring in code outside of that file to be used in that program.
method
Methods are functions in classes.
function
Is used to create a reusable code.
exception
When an error occurs in a method.
from
Used to get code at a particular location.
class variable
Are like constants that can be used by methods within that class. They are usually all caps if they are constant.
inheritance
Is the ability to use all the methods from a particular class.
method overriding
Is when a child class uses a method with the same name and will override the parent method with that method.

What is a class and what can they do?

A class allows you to re-use code. Like a blueprint you can reproduce many instances of it. It is what makes OOP so good. Classes are like databases that you can design them like tables and create fields, like in the movie website we created.
